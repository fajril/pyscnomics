.. pyscnomics documentation master file

Welcome to pyscnomics's documentation!
========================================

pyscnomics is a Python library for calculations and simulations related to
Production Sharing Contracts (PSC). It provides tools for economic analysis,
cost recovery calculations, revenue splitting, and contract optimization.
